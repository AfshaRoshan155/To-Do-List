## Overview
A simple To-Do List web application built using HTML, CSS, and JavaScript. This project allows users to add, manage, and remove tasks easily with a clean and user-friendly interface. It was created to practice DOM manipulation, event handling, and basic frontend development skills.

## Features 
Add Task
  User can write and add new tasks.
Delete Task
  Remove tasks from the list.
Mark as Complete
  Tick/check tasks when done.
Display Task List
  Show all added tasks on screen.
